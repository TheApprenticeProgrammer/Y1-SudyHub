
La population ou ensemble statistique : 
  Est l'ensemble des données sur lesquels vont porter les observations.

Les individus ou unités statistiques :
  Sont les éléments constitutifs de la population/ensemble statistique étudié.

Différence entre recencement et sondage :

  Le rencencement : est utilisé dans certains cas relativement rare, ou l'on peut étudier l'ensemble des individus
                    d'une population relativement à une ou plusieurs caractéristiques (variables statistiques).
                    Exemple : recencement de la population.
                    Dans certains cas ou la population est restreinte le recencement sera facilement réalisable.
                    Exemple : l'ensemble des étudiants d'une classe.

  Toutefois dans la majorité des cas le recencement sera soit impraticable soit il sera trop cher d'étudier l'ensemble
  de la population. On procèdera alors à un sondage.

  Le sondage : c'est losque l'on sélectionne un sous-ensemble de la population/ensemble statistique. Ce sous-ensemble
               s'appelera échatillon. Ensuite nous procéderons à l'étude statistique sur cet échatillon.
               => Attention que l'effectif (taille) de l'échantillon devra être choisi judicieusement.

Les variables statistiques :
  Lorsque l'on fait une étude statistique sur les individus/unités statistiques d'une population/ensemble statistique, c'est
  parceque l'on intéresse à une ou plusieurs caractéristiques particulières de ces individus. Ces caractéristiques sont 
  appelées variables statistiques.
  Exempe : Si l'on étudie l'âge des étudiants d'une classe, la population est constituée des étudiants (de cette
           classe) et chaque étudiant est un individu (unité statistique). La variable statistique étant l'âge
           de chaque étudiant.

Une série statistique ou données brutes :
  Est simplement la représentation sous forme d'un tableau, de la valeur de la variable statistique étudiée pour
  pour chaque individu.

Les modalités d'une variables statistiques :
  Les modalités d'une variable statistiques sont notées : $x_{1}$, $x_{2}$, ... , $x_{i}$. Elles sont les différentes
  valeurs que peut prendre la varaible statistique (il y a autant d'indices numérotés à partir de 1 qu'il n'y a de valeurs
  distinctes de la variable statistique étudiée).

On distingue 4 types de variables statistiques :
  
  => Si les modalités sont des nombres (entiers ou réels), on parle de de variable statistique quantitative :
        - Si les valeurs des modalités sont nombres isolés, on parle de variables quantitatives discrètes.
        Exemple : le nombre d'enfants, le nombre de décès par année, ...
        - Si les valeurs des modalités sont des intervalles de nombres réels, on parle de variables quantitatives continues.
        Exemple : la taille des individus, le salaire des employés, ...

  => Si les modalités ne sont pas des nombres, on parle de variables qualitative :
        - On dira qu'une variable qualitative est nominale, si l'on peut uniquement nommer la modalité.
        Exemple : la couleur préférée, la nationalité, ...
        - On dira qu'une variable qualitative est ordinale, si l'on peut ordonner (selon un ou plusieurs critères) les différentes modalités.
        Exemple : le niveau de satisfaction d'un produit (très bon - bon - moyen - passable - mauvais).

L'effectif total : 
  Est le nombre d'individus pris en compte dans l'étude statistique. On le représente par la lettre N.
  A chaque modalité (valeur) $x_{i}$ de la variable statistique, on associe sont effectif, noté $n_{i}$.

Une distribution statistique :
  On parle de distribution statistique lorque l'on représente les différentes modalités de la série statistique en
  abscisse et leur effectif en ordonnée.
